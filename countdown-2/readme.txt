# Cool Horizontal Countdown Timer

A stylish and cool horizontal countdown timer with a dynamic background, neon glow effect, and subtle pulse animation. This project utilizes HTML, CSS, and JavaScript to create a visually striking countdown experience.

## Features

- Horizontal layout for an elegant and clean design.
- Dark color scheme with a dynamic background gradient.
- Neon glow effect on hover for a cool interactive experience.
- Subtle pulse animation on countdown digits.
- Countdown to a specified target date and time.

## Usage

1. Open the `index.html` file in a modern web browser.
2. Locate the `<script>` tag in the HTML file to set the target date and time within the `updateCountdown` function.
3. Customize the styles in the `<style>` tag to fit your preferences.
4. Hover over the countdown digits to experience the neon glow effect.
5. Countdown digits have a subtle pulse animation for an additional cool factor.

## Files

- `index.html`: The main HTML file containing the structure of the countdown timer.
- `style.css`: The CSS file with styles for the countdown timer.
- `script.js`: The JavaScript file handling the countdown logic and cool effects.
