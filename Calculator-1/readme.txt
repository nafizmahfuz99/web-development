# Aesthetic Calculator

A modern and aesthetically pleasing calculator with a gradient background, rounded buttons, and a sleek design. This project is built using HTML, CSS, and JavaScript to create a visually appealing calculator experience.

## Features

- Stylish and modern design with a gradient background.
- Rounded buttons for a sleek appearance.
- Responsive layout for various screen sizes.
- Basic arithmetic operations (+, -, *, /), percentage (%), and backspace (⌫) functionality.
- Equals (=) button to calculate and display the result.

## Usage

1. Open the `index.html` file in a modern web browser.
2. Use the calculator for basic arithmetic operations:
   - Addition (+)
   - Subtraction (-)
   - Multiplication (*)
   - Division (/)
   - Percentage (%)
3. Clear the display by clicking the "C" button.
4. Use the backspace button (⌫) to delete the last character.
5. Click the "=" button to calculate and display the result.

## Customization

- Feel free to modify the styles in the `<style>` tag to match your preferences.
- You can further enhance the functionality or add additional features as needed.

## Files

- `index.html`: The main HTML file containing the structure of the calculator.
- `style.css`: The CSS file with styles for the calculator.
- `script.js`: The JavaScript file handling the calculator logic and functionality.
