# Dramatic Countdown Timer

A simple countdown timer with a dramatic and visually striking design. This project utilizes HTML, CSS, and JavaScript to create a countdown timer that can be toggled on and off.

## Features

- Dark color scheme for a dramatic effect.
- Pulsating animation on countdown digits.
- Toggle button to start and stop the countdown.
- Subtle styling to enhance the visual appeal.

## Usage

1. Open the `index.html` file in a web browser.
2. Enter the target date and time in the `targetDate` variable within the `updateCountdown` function in the `<script>` tag.
3. Optionally, customize the styles and animations in the `<style>` tag to fit your preferences.
4. Toggle the countdown on and off using the "Toggle Countdown" button.

## Files

- `index.html`: The main HTML file containing the structure of the countdown timer.
- `style.css`: The CSS file with styles for the countdown timer.
- `script.js`: The JavaScript file handling the countdown logic and toggle functionality.
